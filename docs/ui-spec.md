# PortfolioGuard UI Spec

## Landing Page

1. Hero
Dark full-screen canvas with subtle node-grid texture, neon green highlights, and two clear CTAs: `Launch App` and `Watch 90s Demo`.

2. How It Works
Three horizontal glass panels with an icon, short headline, and one concise outcome-focused explanation.

3. Features
Six dense product cards. The emphasis is operational, not marketing-heavy: identity, rules, portfolio analytics, 0G memory, activity proofs, and Uniswap execution.

4. Demo Block
Wide video placeholder on the left, short walkthrough copy and `Try Demo` action on the right.

5. Tech Stack
Sponsor technologies described as active system parts rather than badge wallpaper.

## App Shell

1. Sidebar
Left navigation rail with five destinations: dashboard, chat, portfolio, activity, and settings/rules.

2. Header
Top status bar with wallet connect, Sepolia status, wallet address, and ETH balance.

3. Dashboard
Top row: agent status card.
Middle row: ENS identity card and portfolio overview.
Bottom row: recent activity.

## Chat Screen

1. Large conversation area with assistant and user bubbles.
2. Example prompt chips for demo speed.
3. Right-hand recommendation panel with reasoning bullets and execution button.

## Portfolio Screen

1. Total value and 24h change.
2. Allocation donut chart.
3. Asset list with balance, USD value, and 24h move.

## Activity Screen

1. Chronological cards with title, summary, reasoning, time-ago label, explorer link, and 0G receipt string.

## Settings / Rules

1. Editable rule rows with asset, threshold, amount, cadence, and enabled state.
2. Single save action that persists to 0G-backed memory.
